# 写作页面侧边栏挂件功能说明

## 功能概述

已成功为写作页面添加了侧边栏自定义挂件功能，可以在写作页面的左右两侧显示联系方式、社交媒体链接等信息。

## 实现的功能

### 1. 侧边栏布局
- 写作页面采用三栏布局：左侧边栏 + 主编辑区 + 右侧边栏
- 在小屏幕设备上自动隐藏侧边栏，全屏显示编辑器
- 可以通过设置开关控制是否显示侧边栏

### 2. 预设挂件类型
- **微信号**：显示微信二维码或微信号
- **QQ号**：显示QQ号
- **邮箱**：显示邮箱地址
- **GitHub**：显示GitHub链接（可点击跳转）
- **Twitter**：显示Twitter链接（可点击跳转）

### 3. 自定义挂件
- 支持添加任意自定义链接
- 可以自定义标题、内容和图标
- 支持启用/禁用每个挂件

### 4. 配置界面
- 在写作页面顶部添加了"侧边栏设置"选项
- 可以配置显示哪些挂件
- 可以编辑每个挂件的具体内容

## 使用方法

### 1. 显示/隐藏侧边栏
1. 打开写作页面（/writing）
2. 在顶部找到"侧边栏设置"区域
3. 点击"显示侧边栏"按钮来切换侧边栏的显示状态

### 2. 配置挂件
1. 在"侧边栏设置"区域点击"配置挂件"按钮
2. 可以选择启用/禁用预设的挂件类型
3. 编辑每个挂件的具体内容（如微信号、QQ号等）
4. 添加自定义挂件：
   - 填写标题和内容
   - 点击"添加"按钮

### 3. 示例配置
```
左侧边栏：
- 微信号：your_wechat_id
- QQ号：your_qq_number

右侧边栏：
- 邮箱：your@email.com
- GitHub：github.com/username
- 自定义博客：blog.example.com
```

## 技术实现

### 组件结构
- `SidebarWidgets.tsx`：侧边栏组件，负责显示挂件
- `WidgetConfig.tsx`：配置组件，负责编辑挂件内容
- `writing.tsx`：写作页面，集成侧边栏和配置功能

### 数据流
1. 用户通过配置界面编辑挂件信息
2. 信息保存在组件状态中
3. 侧边栏组件根据配置动态渲染内容
4. 数据不保存到服务器，刷新页面后会恢复默认

## 注意事项

1. 当前配置只保存在浏览器内存中，刷新页面会重置
2. 如需持久化配置，需要后续添加本地存储或服务器存储
3. 图片功能需要自行配置（如显示微信二维码）
4. 外部链接会在新窗口打开
